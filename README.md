# Projeto Unit Tests

# Contexto
Este projeto trata-se de uma série de testes unitarios de funções pré-criadas e outras a serem criadas. Foi utilizado o Jest para criação dos testes.

## Tecnologias usadas

> Desenvolvido usando: JavaScriptES6, Jest

## Testando Localmente

> Clone o Repositório
```bash
git clone git@github.com:Andre-ARS/unit-tests.git
``` 
> Dentro do diretório do projeto, instale as dependencias
```bash
npm install
``` 
> Rode os Testes
```bash
npm test
```
